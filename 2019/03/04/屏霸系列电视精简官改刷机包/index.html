屏霸系列电视精简官改刷机包

乐视屏霸，超4电视uniq系列分体电视官改刷机固件，去除广告，安装限制。适用于电视型号为SHC屏霸，MAX4-55，MAX4-65, UNIQ55，UNIQ65，UNIQ75的电视。

准备工作

一个可以存储刷机文件的U盘

已经格式化FAT32

正常使用的设备（屏霸或者UNIQ系列主机）

下载提供的刷机需要的文件

使用方法

要求

本贴适用的设备

型号：SHC屏霸，MAX4-55，MAX4-65, UNIQ55，UNIQ65，UNIQ75其中之一。

适用现在的系统版本

系统版本必须是EUI6.5、系统版本是EUI6.5、系统版本是EUI6.5。
不是的请自己下载6.5的包，升级或者降级。

附件文件说明

下载后解压：938_shc_0.0.1.7z。解压后的四个文件

1、LetvUpgrade938-SHC.bin 是屏霸系列分体电视的强刷包。文件名固定，该文件是单recovery分区的强刷包。
2、UPDATE-Recovery-After-Install 是刷入recovery后需要马上刷入的包，可以对系统恢复recovery进行处理。
3、lemod_shc_0.0.1.zip 是制作好的卡刷包，系统精简处理版本号是eui6.5。
4、UPDATE-SuperSU-v2.82** 是supersu的卡刷包，用以获取root权限

刷机步骤

第一步：

A、B方案任选，A不行的可以试试B。注意在用A、B方案刷入recovery后，电视不能开机进系统、否则重新刷入。

A：分体电视设备待机下，设置、音量加、音量减、频道减、频道加、电源键。稍等几秒，电视进度条一闪而过，设备重新待机。此时recovery已经刷好。
B：分体电视设备断电下，按设备机身确定键保持不动，设备通电。稍等几秒，电视进度条一闪而过，设备重新待机。此时recovery已经刷好。

A、B方案任选，A不行的可以试试B。

A：分体电视设备待机下，设置、音量加、音量减、频道加、频道减、电源键。稍等几秒，电视进入recovery。
B：分体电视设备断电下，按设备机身音量加保持不动，设备通电。稍等几秒，电视进入recovery。

第二步：

####

1、将光标移动到wipe data/factory reset，选中确定，接着选yes。（就是通常所说的恢复出厂）
2、将光标移动到wipe cache partition，选中确定，接着选yes。（就是通常所说的恢复出厂）
3、将光标移动到apply update from external storage，选中确定，然后会出现U盘列表，一般情况下是no name。
4、选中你的U盘，确定进入你的U盘的文件列表。依次序选中UPDATE-Recovery-After-Install 、lemod_shc_0.0.1.zip 、UPDATE-SuperSU-v2.82**刷入。
举例：选中UPDATE-Recovery-After-Install确定，选yes，开始刷入。刷入完成后会重新弹出菜 单，此时重复步骤3/4。
5、三个文件刷完后。在弹出的菜单处选择，reboot system now。电视重新待机，遥控开机即可。

特色：

1、 去除了乐视的开机广告
2、 去除乐视桌面推送的广告
3、 去除乐视的不实用桌面插件
4、 只保留信号源、应用、轮播桌面插件
5、 应用桌面置顶的10个应用修改为旧版本5应用
6、 5应用分别为云视听电视猫、星火直播、银河奇异果、文件管理、浏览器
7、 去除了广电对应用安装的管控
8、 修改电视点击屏保为风景图片
9、 采用官方eui6.5修改，zip刷机包降低风险

玩机小技巧

1、桌面管理里，你可以添加轮播了。添加后如果想删除，就只能清楚桌面数据了。
2、开机你可以自定义开机视频啦，只需要将你的开机视频改名字为video。没有后缀的，放入到机身存储的目录下（在ES文件管理器里的主目录、乐视文件管理器的本地存储），新建fastplay文件夹，将此文件放进去就可以了。
