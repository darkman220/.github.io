超3电视928芯片官改刷机包-ui6.0

使用方法

要求

本贴适用的设备

电视型号为超3 X50、超3 X55、超3 X55 pro、超3-X65、X65、X40S、X43S、X49、超级电视 Max120、S40升级主板s40pro、S50升级主板s50pro的电视其中之一。

适用现在系统版本

本zip刷机包为去除底层的完整系统包。wipe(恢复出厂设置)后刷入

附件文件说明

下载后解压：928_60_072_0822_beta2.7z。解压后的四个文件

1、LetvUpgrade928_10M.bin dipcore制作，刷机时需更正为LetvUpgrade928.bin,该文件是单recovery分区的强刷包。
2、LetvUpgrade928_9M.bin 蓝同学制作，刷机时需更正为LetvUpgrade928.bin,该文件是单recovery分区的强刷包。
3、928_60_072N_0822_beta2.zip 是制作好的卡刷包，系统精简处理版本号是eui6.0。

刷机步骤

刷入适合自己机型的recovery

recovery的选择

1080P屏幕的设备（X40S、X43S、S40升级主板s40pro、S50升级主板s50pro等），只能使用 LetvUpgrade928_9M.bin
4K屏幕的设备（超3 X50、超3 X55、超3 X55 pro、超3-X65、X65、超级电视 Max120等），可以使用LetvUpgrade928_10M.bin和LetvUpgrade928_9M.bin，推荐使用LetvUpgrade928_10M.bin
使用LetvUpgrade928_10M.bin的recovery刷机时只能使用4k刷机方案。（有recovery界面，手动操作。可以进入recovery进行刷机前的恢复出厂设置）
使用LetvUpgrade928_9M.bin的recovery刷机时能使用1080P刷机方案。（无recovery界面，全自动刷机。刷机前，刷机后为保证稳定需恢复出厂设置）

刷入recovery (文件名必须为LetvUpgrade928.bin)

A、B方案任选，A不行的可以试试B。
A：设备待机下，
超级遥控器使用：设置、音量加、音量减、频道减、频道加、电源键。稍等几秒，电视进度条一闪而过，设备重新待机。此时recovery已经刷好。
39键遥控器使用：3、6、9、5、开机键。稍等几秒，电视进度条一闪而过，设备重新待机。此时recovery已经刷好。
B：设备断电下，按设备机身确定键保持不动，设备通电。稍等几秒，电视进度条一闪而过，设备重新待机。此时recovery已经刷好。

刷机

根据你刷入的recovery，选择刷机方案。

4K刷机方案

#####
A、B方案任选，A不行的可以试试B。
A：设备待机下
超级遥控器：设置、音量加、音量减、频道加、频道减、电源键。稍等几秒，电视进入recovery。
39遥控器：3、6、9、1、开机键。稍等几秒，电视进入recovery。
B：设备断电下，按设备机身音量加保持不动，设备通电。稍等几秒，电视进入recovery。

#####
1、将光标移动到wipe data/factory reset，选中确定，接着选yes。（就是通常所说的恢复出厂）
2、将光标移动到wipe cache partition，选中确定，接着选yes。（就是通常所说的恢复出厂）
3、将光标移动到apply update from external storage，选中确定，然后会出现U盘列表，一般情况下是no name。
4、选中你的U盘，确定进入你的U盘的文件列表。选中928_60_072N_beta2.zip确定，选yes，开始刷入。刷入完成后会重新弹出菜单。
5、刷完后。在弹出的菜单处选择，reboot system now。电视重新待机，遥控开机即可。

1080P刷机方案

将电视恢复出厂设置后关机。将928_60_072N_beta2.zip重命名为update.zip后放入U盘。
A、B方案任选，A不行的可以试试B。
A：设备待机下
超级遥控器：设置、音量加、音量减、频道加、频道减、电源键。稍等几秒，电视显示进度条，开始自动升级。
39遥控器：3、6、9、1、开机键。稍等几秒，电视显示进度条，开始自动升级。
B：设备断电下，按设备机身音量加保持不动，设备通电。稍等几秒，电视显示进度条，开始自动升级。
重新开机进入系统后需要回复出厂设置操作。如果不能进入系统可使用39遥控器3692开机，进行恢复出厂设置操作

特色：

1、 去除了乐视的开机广告
2、 去除乐视桌面推送的广告
3、 去除乐视的不实用桌面插件
4、 只保留信号源、应用、轮播桌面插件
5、 应用桌面置顶的5个应用修改为云视听电视猫、星火new直播、银河奇异果、文件管理器、浏览器
6、 去除了广电对应用安装的管控
7、 修改电视点击屏保为风景图片
8、 采用官方eui6.0修改，zip刷机包降低风险

玩机小技巧

1、桌面管理里，你可以添加轮播了。
2、开机你可以自定义开机视频啦，只需要将你的开机视频改名字为video。没有后缀的，放入到机身存储的目录下（在ES文件管理器里的主目录、乐视文件管理器的本地存储），新建fastplay文件夹，将此文件放进去就可以了。
3、如何修改应用界面，应用界面位置为”/system/plugin/com.stv.plugin.app.apk”，可以使用别人制作好的应用插件替换我内置的插件。
4、需要用谷歌框架的请自行刷入谷歌补丁

常见问题

1、刷机后恢复出厂设置提示系统损坏

解决办法：重新刷入recovery

2、进入10M recovery后遥控不能操作

解决办法：电视关机，带着遥控器去别的房间一会儿，重新进入recovery操作

3、无法刷入recovery，遥控操作完就开机进入系统

解决办法：遥控器按得不能太快，也不能太慢，最好是按一个键停顿一下。
